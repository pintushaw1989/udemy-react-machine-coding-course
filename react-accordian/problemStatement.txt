Problem Statement

-> Build a reusable and accessible React Accordion component 
   that displays a list of expandable/collapsible items 
   with smooth open/close animations.
   (only one item open at a time)
-> Then extend it - to support Multi Select mode and 
   Provide Expand All and Collapse All actions in multi-select mode.
   (multiple items can be open simultaneously)
-> Then extend it - Allow specifying default open items.
-> Handle edge cases such as an empty list or invalid input by 
   displaying "No items available".


Improvements
-> Handle dynamic content height changes 
   (e.g., images or async-loaded content).
-> Ensure accessibility using 
   semantic buttons and appropriate ARIA attributes.
-> Separate business logic using a custom hook 
   and optimize rendering where appropriate.
